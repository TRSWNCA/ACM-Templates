# ACM 模板库

以模板题为主进行模板的扩充

## 使用

克隆到本地：
```bash
$ git clone https://github.com/TRSWNCA/ACM-Templates
```

为自动提交提供权限：

```bash
$ chmod +x ACM-Templates/submit.sh
```

添加好代码后提交：

```bash
$ ./submit.sh
```
